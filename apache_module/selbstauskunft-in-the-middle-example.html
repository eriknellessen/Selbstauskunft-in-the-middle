Selbstauskunft in the middle example web page

This is an example web page to demonstrate the concept of the scenario "Selbstauskunft in the middle".

Did you already connect the card reader on the client computer to the virtual card reader on the web server? If you did not already do it, remember to do it before clicking the button.

When you click the button, the web server will start the Selbstauskunft procedure to read the data from your nPA. You will have to enter your PIN on your card reader. If your first name indicates that you are allowed to see the secret, it will show you the secret. Else, it will show you the result of the Selbstauskunft procedure.
